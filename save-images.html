Follow these instructions to add your logo and fan images

All code has been updated to reference the new image files. You just need to save the 3 images with the correct filenames in the root directory of your project.

Step 2: Save the Images

✅ How to Save Each Image:

  1. Right-click on the first image attachment (red Prodigy logo)
  2. Select "Save Image As..." or "Download Image"
  3. Navigate to your Prodigy website project folder (the root directory)
  4. Save the file with the exact name: logo.png
  5. Repeat for the second image (white fan) → save as: HSFan.png
  6. Repeat for the third image (black fan) → save as: MSFan.png

Step 3: File Location

All 3 images must be saved in the root directory of your project:

📁 Correct File Structure:

prodigy-website/
├── index.html
├── script.js
├── logo.png          ← Save here
├── HSFan.png         ← Save here
├── MSFan.png         ← Save here
├── favicon.svg
├── logo.svg (old, can be deleted)
├── styles/
│   ├── style.css
│   └── tools.css
└── tools/
    ├── fan-selector.html
    ├── esp-calculator.html
    └── airflow-calculator.html

Step 4: Verify Everything Works

🔍 Testing Checklist:

  1. Open index.html in your browser
  2. Check header - Prodigy logo should appear (55px width, red square with white P)
  3. Scroll to footer - Logo should appear there too
  4. Navigate to /tools/fan-selector.html
  5. Select "Plastic (HS)" - White fan image should display
  6. Select "Metal (MS)" - Black fan image should display
  7. Check all 3 tool pages - Logo should appear in headers and footers

📥 Quick Reference

Image filenames (must be exact):

logo.png HSFan.png MSFan.png
