# Documentation Folder

## Catalog PDF

Place your **Catalog.pdf** file in this directory.

The "Download Catalogue" button throughout the website is already configured to download this file.

### File Requirements:
- **File Name:** `Catalog.pdf` (exact name, case-sensitive)
- **Location:** `/Documentation/Catalog.pdf`

Once you add your PDF file here, the download functionality will work automatically across all pages:
- Main navigation menu
- Product pages
- Calculator tools
- All other pages with the "Download Catalogue" button

No code changes needed - just drop your PDF file in this folder!
